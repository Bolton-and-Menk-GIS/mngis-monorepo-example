(async) filterAsync(array, callbackfn) → {Array.<Any>}

Source:
  • index.js, line 107
See:
  • https://stackoverflow.com/a/53508547/3005089
performs an Array.prototype.filter function on an array using an asynchronous function
Example
async function isEven(n) {
  await setTimeout(()=> null, 250);
  return !(n % 2)
}
const vals = [2, 4, 3, 7, 8, 9, 10, 11];
await filterAsync(vals, isEven);
// [2, 4, 8, 10]
Parameters:
Name Type Description
array Array.<Any> input array of items to run against an async function
callbackfn function an asynchronous function
Returns:
a filtered array of the outputs from the asynchronous function for each item in array

nestedProp(p) → {function}

Source:
  • index.js, line 158
fetches nested properties from an object
Examples
// simple example with a single object (check for editable geometry)
const layerConfigs = [
{
   name: 'Map Notes',
   edits: {
     add: true,
     update: {
       attributes: true,
       geometry: true
     },
     delete: true
   }
 }, {
   name: 'Storm Manholes',
   edits: {
     add: true,
     update: {
       attributes: true,
       geometry: false
     },
    delete: false
   }
 }]
const canEditGeometry = nestedProp(['edits', 'update', 'geometry'])(layerConfig); // true, same as calling layerConfig.edits.update.geometry
// can also be used for one level deep
const edits = nestedProp('edits')(layerConfig) //  { add: true, update: { attributes: true, geometry: true }, delete: true }
// find all editable geometries in a list
const layerConfigs = [
{
   name: 'Map Notes',
   edits: {
     add: true,
     update: {
       attributes: true,
       geometry: true
     },
     delete: true
   }
 }, {
   name: 'Storm Manholes',
   edits: {
     add: true,
     update: {
       attributes: true,
       geometry: false
     },
    delete: false
   }
 }]

// now filter the array to only include editable geometries (1 result, curried function will automatically take each item)
layerConfigs.filter(nestedProp(['edits', 'update', 'geometry'])); // {name: 'Map Notes', ...}

a function to call for the property, you would then use this call on an object: nestedProp(['edits', 'add'])()

toArray(obj) → {Array}

Source:
  • index.js, line 112
Guarantees that the input is an Array. If the input is not an Array, it will be added to one.
Examples
// returns [1];
const layerName = 'Cities';
const layerNames = toArray(layerName); // -> ['Cities']
// can also be used to validation and ensure the output is an array
const oidList = [1, 2, 3];
// returns the original object, because it is already an Array: [1, 2, 3]
toArray(oidList) // -> [1, 2, 3]
Parameters:
Name Type Description
obj Object | Array | Number | String Object of any type to put into an array.
Returns:
an Array or the original object if input was an Array. Will put any inputs into an Array, or can be used to ensure something is an Array.
Type
Array
